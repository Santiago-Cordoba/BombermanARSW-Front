# BombermanARSW-Front

Como Jugador,

Quiero establecer la duración de la partida, cantidad de jugadores, cantidad de poderes, cantidad de bloques 

Para Poder jugar una partida personalizada.



  1. El sistema debe permitir seleccionar la duración de la partida antes de iniciarla (2,5,10 minutos).
  2. Debe ser posible establecer el número de jugadores que participarán en la partida (1-4).
  3. Debe ser posible establecer el número de bloques que se quieran en la partida (1-30).
  4. Debe ser posible establecer el número de vidas presentes para el jugador (1-10).
  5. La configuración seleccionada debe aplicarse correctamente antes de que la partida comience.
  6. Los valores configurados deben mostrarse claramente en la interfaz antes de confirmar la partida.
  7. Si un jugador intenta establecer valores fuera de los límites permitidos, el sistema debe mostrar una advertencia y evitar la configuración incorrecta.
  8. El jugador debe poder seleccionar en la configuración el numero de vidas para cada jugador con las que desea que se juegue la partida.
  9. Todas estas configuraciones deben ser escogidas por el usuario que esta hosteando la partida.

Como jugador,
Quiero poder iniciar un partida
Para empezar a jugar.


  1. El jugador puede seleccionar "Iniciar partida" desde el menú principal y lo redirige a la vista de seleccionar las configuraciones.
  2. El juego carga la partida con las respectivas configuraciones que se escogieron.
  3. El jugador puede empezar a realizar todas las acciones de juego una vez iniciado el juego.
  4. La partida se inicia de manera correcta con todos los jugadores que se configuraron inicialmente.
